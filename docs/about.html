About this website

This site was created by DIGIT students for DIGIT students and anyone interested in the DIGIT program. The site was designed with sustainable development in mind. We want this site to be easy to edit and revise and even completely redesign by cycles of future students.

The site was first created in version 1.0 in 2020 by Mia Borgia. The current version is 2.0. The site development and documentation proceeds on GitHub.

The following students have contributed to the design and development of this website:

  • Felipe Bassi:
    led a total redesign of this website from the original version to version 2.0 in 2022.
  • Mia Borgia:
    created the original version of this website in 2020.
  • Julian Giles:
    created the SVG infographic of the course roadmap in 2022.
  • Yuying Jin:
    redesigned this "About" page in 2022.
  • William Stiller:
    redesigned the Project page for the interactive tile view of version 2.0 in 2022.
